General

  • infix functions e.g. 1+2 = (+) 1 2 where + is infix
    • and for (+a) b = b + a = (+)b a, a is the second argument and the following a
  • functions are always applied to exactly one argument at a time

Lazy evaluation

To evaluate (t1t2)

  1. evaluate t1
  2. if the result of t1 is (λx.u), return u[x/t2]
  3. Then evaluate the result of the substitution
    To evaluate (λx.t) return it unchanged

Eager evaluation

To evaluate (t1t2)

  1. evaluate t1, then evaluate t2 to t2′
  2. if the result of t1 is (λx.u), return u[x/t2′]
  3. Then evaluate the result of the substitution
    To evaluate (λx.t), evaluate t to t′ and return (λx.t′)

Common mistakes/special care

  • incorrect parenthesis at the beginning: e.g. λx.x(λy.xy)≢(λx.x)(λy.xy)
  • substitution removes the lambda (λx.x)(λy.xy)⇝̸λ(λy.xy).λy.xy but just λy.xy
  • in lazy evaluation there is no evaluation under lambdas
  • in eager evaluation, need to evaluate argument before substitution
