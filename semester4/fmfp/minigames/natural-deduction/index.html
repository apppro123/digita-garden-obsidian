Natural Deduction

Abstract example of a formal proof

  • rules are used to construct derivations under assumptions
  • proof is a derivation whose root has no assumptions (at the root on the left there is nothing)
  • derivations are trees (normally done from bottom to top)

Propositional logic

Syntax

  • Is a bird a formula? No, bc. it's about the smallest set (inductive definition), so there is not other stuff there!

Semantics

  • valuation σ:V→{True,False} is a function mapping variables to truth values (truth assignment).
    • valuations are simple kinds of models (interpretations)
    • Let Valuations be the set of valuations.
  • Satisfiability:
  • Note that σ⊭⊥ for every σ∈ Valuations: meaning falsity is never satisfied (as it should bebn )
  • a formula A∈LP is satisfiable if σ⊨A, for some valuation σ
    • valid (a tautology) if it holds for all valuations σ
  • semantic entailment: A1,..An⊨A if for all σ, if σ⊨A1,…,σ⊨An then σ⊨A

requirements for deductive system

  • syntactic entailment ⊢ (derivation rules) semantic entailment ⊨ (truth tables) should agree
  • For Γ≡A1,…,An some collection of formulae for which holds:
    • soundness: If Γ⊢A can be derived, then Γ⊨A
    • completeness: If Γ⊨A, then Γ⊢A can be derived
  • (desirable, but not necessary) Decidability: What is complexity of determining:
    • If a proposition A is satisfied by a valuation σ? => linear
    • If A is satisfiable? => exponential to check all possible evaluation and it's NP-complete
    • If A is a tautology? => it's in Co-NP

First-Order Logic

Syntax

  • signature consists of a set of function symbols F and a set of predicate symbols P (and their arities e.g. fk to indicate function/predicate symbol f has arity k∈N)
    • constants are 0−ary function symbols
  • Let V be a set of variables
  • Term, the terms of first-order logic, is the smallest set where
  1. x∈Term if x∈V and
  2. fn(t1,…,tn)∈Term if fn∈F and ti∈Term for all 1≤i≤n
  • Form, the formulae of first-order logic, is the smallest set where
  1. ⊥∈Form
  2. pn(t1,…,tn)∈Form if pn∈P and tj∈Term, for all 1≤j≤n
  3. A∘B∈Form if A∈Form, B∈Form, and ∘∈{∧,∨,→}
  4. Qx.A∈Form if A∈Form,x∈V and Q∈{∀,∃}
  • bound/free
    • names of bound variables are irrelevant, they just encode the binding structure
    • α-conversion: renaming bound variables is okay
  • Operators: omitting paranthesis:
    • Binary operators:
      • ∧ stronger than ∨ stronger than →
      • associates: → to the right while ∧ and ∨ do to the left
    • negation stronger than binary
    • quantifiers extend to the right as far as possible i.e. end of line or )

Semantics

  • structure is a pair S=⟨US,IS⟩ where US is a nonempty set, the universe, and IS is a mapping where
    1. IS(pn) is an n-ary relation on US, for pn∈P
    2. IS(fn) is an n-ary (total) function on US, for fn∈F
      As shorthand, write pS for IS(p) and fS for IS(f)
  • an interpretation is a pair I=⟨S,v⟩, where S=⟨US,IS⟩ is a structure and v:V→US a valuation.
  • The value of a term t under the interpretation I=⟨S,v⟩ is written as I(t) and defined by
    1. I(x)=v(x), for x∈V and
    2. I(f(t1,…,tn))=fS(I(t1),…,I(tn))

    • replace in a formula all occurences of a free variable x with some term t e.g.
    • substitution must avoid capture of variables: if they would have the same name, then we have to first rename bound variables with α-conversion
